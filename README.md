📌 Fila Zero - Sistema de Login
📖 Descrição

Este é um projeto simples de console em C# que simula um sistema de login.
O usuário insere um nome de usuário e uma senha, e o sistema verifica se as credenciais estão corretas.

🚀 Funcionalidades
Interface simples no terminal
Entrada de usuário e senha
Validação básica de login
Mensagem de sucesso ou erro
🧠 Lógica do Sistema

O sistema funciona da seguinte forma:

Exibe uma tela de título
Solicita:
Nome de usuário
Senha
Compara com valores pré-definidos:
Usuário: admin
Senha: senha123
Retorna:
✅ Login bem-sucedido
❌ Login falhou

⚙️ Como Executar
Abra o terminal ou Visual Studio
Crie um projeto Console em C#
Substitua o conteúdo pelo código acima
Execute o programa
🛠️ Tecnologias Utilizadas
C#
.NET (Console Application)
📌 Observações
Este projeto é apenas uma simulação, não possui segurança real.
As credenciais estão fixas no código.
Ideal para aprendizado de lógica e entrada de dados no console.
💡 Possíveis Melhorias
Ocultar a senha ao digitar
Criar sistema de cadastro
Validar múltiplos usuários
Integrar com banco de dados
